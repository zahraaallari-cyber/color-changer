# 🎨 Color Changer App

## 📌 Description
Color Changer App is a simple desktop application built using Python and PySide6.  
It allows users to dynamically change the background color of the window using RGB sliders or preset color buttons.

This project demonstrates fundamental concepts of GUI (Graphical User Interface) development, including user interaction, event handling, and real-time UI updates.

---

## 🎯 Features
- 🎚️ RGB sliders to control Red, Green, and Blue values
- 🎨 Real-time background color update
- 🔘 Preset color buttons (Red, Green, Blue)
- 🔄 Reset button to restore default color
- 📝 Displays current RGB color value

---

## 🛠️ Technologies Used
- Python
- PySide6 (Qt for Python)
