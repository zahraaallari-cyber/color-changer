# 🎨 Color Changer App

## 📌 Description
Color Changer App is a simple desktop application built using Python and PyQt6.  
It allows the user to change the background color of the window by clicking a button.

---

## 🎯 Features
- Simple and clean user interface
- Button to change background color
- Instant visual update
- Easy to use

---

## 🛠️ Technologies Used
- Python
- PyQt6

---

## ▶️ How to Run

1. Make sure Python is installed
2. Install PyQt6:
   pip install PyQt6

3. Run the program:
   python main.py

---

## 💡 How It Works
- The application creates a window using PyQt6
- A button is added to the window
- When the user clicks the button, the background color changes
